# AI Vocabulary Reference

Words and phrases that appear disproportionately in AI-generated text. Avoid these or use sparingly with the suggested alternatives.

## Tier 1: Strong AI Signals (Avoid Entirely)

These words are strong indicators of AI authorship. Find alternatives.

| Word | Alternatives |
|------|-------------|
| delve | examine, explore, look at, dig into |
| tapestry | mix, combination, collection |
| vibrant | active, busy, lively |
| landscape | field, area, scene, situation |
| realm | field, area, domain |
| embark | start, begin |
| foster | encourage, support, develop |
| underpinning | basis, foundation, support |
| pivotal | important, key, critical |
| holistic | complete, whole, comprehensive |
| multifaceted | complex, varied |
| nuanced | subtle, detailed, complex |
| intricate | complex, detailed |
| leverage | use, apply |
| utilize | use |
| encompasses | includes, covers |
| groundbreaking | new, original, innovative |
| transformative | significant, major |
| cornerstone | foundation, basis, key part |
| testament | proof, evidence, sign |
| epitomize | represent, embody, typify |
| spearhead | lead, start |
| culminate | end, result in |

## Tier 2: Moderate AI Signals (Use Sparingly)

These words aren't wrong but appear too often in AI text. One per piece maximum.

| Word | When Acceptable |
|------|-----------------|
| comprehensive | When describing scope that truly is complete |
| enhance | In technical contexts about improvement |
| crucial | When something is genuinely critical |
| underscore | When something is being emphasized |
| showcase | When displaying/presenting is literal |
| navigate | When movement or wayfinding is literal |
| robust | In technical specs |
| seamless | When integration is genuinely smooth |
| streamline | When process simplification is the topic |
| dynamic | In technical contexts (dynamic typing, etc.) |
| innovative | Only with specific evidence of novelty |

## Tier 4: Filler Intensifiers (Delete)

These add no meaning. Delete them.

- truly
- really
- very
- highly
- deeply
- extremely
- incredibly
- remarkably
- profoundly
- vastly

## AI Phrases to Avoid

Multi-word patterns that signal AI:

| Phrase | Alternative |
|--------|-------------|
| "It's important to note that" | Delete, just state the thing |
| "It's worth noting that" | Delete, just state the thing |
| "In conclusion" | Delete or use "To sum up" |
| "First and foremost" | "First" or delete |
| "Last but not least" | "Finally" or delete |
| "At the end of the day" | Delete |
| "In today's world" | Delete or specify the era |
| "In this day and age" | Delete |
| "It goes without saying" | Then don't say it |
| "Needless to say" | Then don't say it |
| "A testament to" | "shows" or "demonstrates" |
| "Plays a crucial role" | Specify what it does |
| "A wide range of" | Specify the range |
| "A myriad of" | "many" or specify count |
| "Throughout history" | Specify the time period |
| "Stands as a" | "is" |
| "Serves as a" | "is" |

## Opening Phrases to Avoid

Don't start paragraphs or sections with:

- "In the realm of..."
- "When it comes to..."
- "In terms of..."
- "With regard to..."
- "As we navigate..."
- "In an era of..."
- "In the context of..."

Just start with the subject.

## Closing Phrases to Avoid

Don't end with:

- "...continues to inspire"
- "...remains relevant today"
- "...stands as a testament to"
- "...paved the way for"
- "...shaped the landscape of"
- "...left an indelible mark"
- "...challenges and opportunities ahead"

End with specific facts or clear conclusions.
