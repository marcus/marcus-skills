Folio

v1.0 Design System

Where the image speaks first.

Folio is the design language for high-end photography portfolios. It draws from the visual conventions of Gagosian, Magnum Photos, and Fotografiska — spaces where the photograph is always the protagonist, and the interface exists only to frame it. Every token, every ratio, every omission serves that hierarchy.

Thesis

Photography portfolios fail when the design competes with the work. Folio inverts the typical relationship between interface and content. The UI is the wall paint, the frame, the negative space between prints. It does not interpret. It does not editorialize. It recedes — and in receding, it elevates.

Principles
01

Image First

Every design decision defers to the photograph. Typography, color, and UI exist to frame, not compete. If a visual element draws the eye before the image does, it has failed.

02

Deliberate Silence

White space is not emptiness. It is the wall that gives the print room to breathe. Generous margins and vertical rhythm create the contemplative pace that serious photography demands.

03

Sharp Precision

No rounded containers. The rectangle is the frame. Geometry signals intention. Every edge is deliberate, every alignment exact. The sharpness of the interface mirrors the sharpness of the print.

04

Quiet Authority

Light weights. Restrained color. Confidence expressed through what is withheld. A single terracotta accent against near-achromatic neutrals. The system never raises its voice.

Color

Color palette

Near-achromatic with warm undertones. A single deep terracotta accent. Photography dominates; color stays subordinate. No cool grays — they compete with images.

Typography

Type specimen

Crimson Pro for display. Instrument Sans for body and UI. Light weights at large scale. ALL-CAPS labels with wide tracking for metadata, dates, and navigation. The serif carries gravitas; the sans does the work.

Spacing & Layout

Spatial system

Built on a 4px base grid. Photography sites use extreme vertical whitespace — section spacing starts at 96px and reaches 160px for hero-level breathing room. The scale is non-linear; jumps between larger values create the contemplative pace of a gallery wall.

Motion

Movement language

Restrained. No parallax, no scroll-jacking. Photography is static; the UI should be too. Custom easing curves only — never CSS presets. Motion should be felt, not watched.
