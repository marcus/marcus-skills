States

Disabled and loading

Disabled buttons reduce opacity and remove pointer events. Loading buttons show a spinner and block interaction.

Scale

Size variants

Use small buttons in dense UI like table rows. Large buttons anchor hero sections and confirmation dialogs.

Destructive

Danger variant

Reserve the danger variant for irreversible actions. Always pair with a confirmation step.
