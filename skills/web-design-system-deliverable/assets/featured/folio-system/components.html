Buttons

Priority hierarchy through shape, fill, and weight. Primary actions use the pill signature; secondary and ghost recede intentionally.

Priority Ladder

Primary: pill, accent fill, weight 500, ALL-CAPS tracking. Secondary: 4px radius, 1px border. Ghost: text only, underline on hover. Danger: 4px radius, danger fill.

Editorial & Navigation Variants

Editorial: Crimson Pro italic, no container, underline grows from center on hover. Arrow: ghost with → glyph, gap widens 6px → 12px on hover.

Sizes

Small: 11px / 8px 16px. Medium: text-xs / 12px 24px. Large: text-sm / 16px 32px. ALL-CAPS with widest tracking at every size.

Hover State

Primary darkens to accent-600. Secondary border strengthens. Ghost gains underline. Transition: 150ms cubic-bezier(0.22, 0.61, 0.36, 1).

Focus Visible

2px accent ring with 2px offset. Visible on keyboard navigation only.

Active (Pressed)

Scale 0.98 on press. Subtle, tactile feedback.

Disabled

Opacity 0.4, pointer-events: none, cursor: not-allowed.

Loading

Text hidden, centered spinner. Pointer events disabled during loading.

Form Controls

Bottom-line inputs: quiet at rest, revealing on interaction. No visible containers until focus. ALL-CAPS labels with widest tracking follow the gallery convention.

Badges & Tags

Three badge approaches calibrated to visual weight. Dot-indicator by default, filled pill reserved for the most critical status, and typography-only for editorial contexts.

Dot-Indicator Badges (Default)
Draft Exhibition Published Edition Selling Sold Out Featured

7px colored dot + text, no container. Uppercase, widest tracking. Color conveys status; the dot provides the only chromatic weight.

Alerts

Visual escalation mapped to actual urgency. Info is bare text, success uses a left accent, warning gets a top border, and only danger earns a full colored container.

Exhibition submission received
The portfolio for "Traces of Light" at Foam Amsterdam has been submitted. Review typically takes 5 – 7 business days.
Print edition published
"Untitled #7, 2024" is now available. Archival pigment print, edition of 15. Listed at $2,400 per print.
Edition running low
"The Quiet Hour, Print III" has 2 of 12 editions remaining. The availability status on the portfolio page may need updating.
The file couldn't be uploaded
"DSC_4827.tiff" exceeds the 200 MB limit. The file will need to be compressed or resized before it can be accepted.

Severity tiers: Info = bare text (no container). Success = 2px left accent + neutral bg. Warning = 3px top border + neutral bg. Danger = full tinted container with border (the only alert that earns it).
