Font pairing rationale

Instrument Serif

Editorial warmth and personality. Excellent x-height for readability at display sizes. The italic cut brings a sense of considered elegance to headlines. Used for hero, display, and title-lg styles.

Inter

Geometric clarity with superior screen rendering. Wide weight range from 400-700 handles every context from body copy to bold headings. Optimized for UI with tabular figures and distinct letterforms.

JetBrains Mono

Purpose-built for code, data, and technical content. Consistent character width enables perfect alignment in tables and code blocks. Distinct zero and capital O prevent confusion.
