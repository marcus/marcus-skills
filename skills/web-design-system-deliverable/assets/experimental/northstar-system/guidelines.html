Usage guide

Rules that keep the system coherent as more teams touch it.

Use these principles to apply the system consistently across marketing pages, product surfaces, internal tooling, and launch materials.

Core principle

Make hierarchy visible before you add decoration.

Spacing, type scale, and surface contrast should explain the interface before color or motion adds emphasis.

Principles

Five rules for an agency-grade system.

01

Lead with type

Use the display serif sparingly for authority and body sans for pace. Never flatten both into the same voice.

02

Use accent with restraint

Accent color marks decision points and signature moments. It should not become a wallpaper.

03

Keep spacing on one rhythm

Layouts should feel measured. Skip arbitrary in-between values that break vertical cadence.

Do
  • Map brand palette into semantic roles before styling components.
  • Document why each variant exists and where it should not be used.
  • Use visible focus indicators and reduced-motion fallbacks from the start.
  • Show components in realistic page contexts, not only isolated chips.
Avoid
  • Creating one-off colors for individual campaigns or screens.
  • Letting surface shadows replace actual hierarchy and spacing.
  • Adding dark mode by inverting every value without role review.
  • Publishing components without content, motion, and accessibility guidance.
