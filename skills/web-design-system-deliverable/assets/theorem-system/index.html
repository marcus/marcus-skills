Theorem

v1.0 Design System

A design system built on the premise that precision is the highest form of expression.

Theorem provides the typographic, spatial, and chromatic foundation for an agentic AI platform. It borrows from editorial tradition rather than technology convention. Every token is deliberate. Every omission is intentional.

Principles
1

Precision over decoration

Every element earns its place. White space is the primary design material. If something can be removed without loss of meaning, it should be. The system favors structural clarity over ornamental richness.

2

Typography carries the brand

The tension between serif and sans, the hierarchy of weight and tracking, the editorial spacing between elements. These do the visual work before color enters the conversation. Type is not a vehicle for content. It is the content.

3

Restraint signals confidence

One accent color. Subtle borders. Motion you feel but don't notice. The system resists the impulse to announce itself. It communicates authority through what it withholds, not what it displays.

Motion

Movement language

Custom easing curves, never CSS presets. Motion should be felt, not watched. Hover interactions at 160ms. Transitions at 280ms. Entrances at 450ms. Stagger at 60ms per item.

Interaction signatures
Hover lift
Interactive elements scale up subtly on hover. Barely perceptible, but enough to confirm the element is alive. Applied to cards, buttons, and linked containers.
transform: scale(1.015)
Press sink
On mousedown, elements contract slightly. The tactile response bridges the gap between cursor and interface. Released on mouseup with spring easing.
transform: scale(0.98)
Stagger entrance
Lists and grids reveal items sequentially with a 60ms delay between each. The cascade draws the eye through content in reading order without demanding attention.
transition-delay: n * 60ms
