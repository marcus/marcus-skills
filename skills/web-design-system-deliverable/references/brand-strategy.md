# Brand Strategy

Run strategy before touching tokens or components. This phase produces the positioning, personality, and visual direction that everything else builds on.

## 1. Brand Audit Framework

Assess the current state before proposing anything new.

### Questions to Answer

- What does the brand look like today? Collect screenshots, assets, current guidelines.
- What is working? What feels dated, inconsistent, or misaligned with business goals?
- Who are the three closest competitors? How do they present visually?
- Where is the whitespace? What visual territory is unclaimed in this category?
- What business shift is driving the rebrand or new system? Growth, repositioning, new audience, credibility gap?

### Audit Deliverable

Produce a short document with:

- Current state summary (3-5 sentences, with screenshots if available)
- Competitor landscape table: name, visual character, tone, weakness
- Whitespace opportunities (2-3 underexploited positions in the market)
- Tensions to resolve (e.g., "brand says premium but typography says template")

## 2. Positioning Model

Define where the brand sits and where it needs to move.

### Positioning Matrix Template

```
             Conservative
                  |
     Established  |  Institutional
                  |
Accessible -------+------- Premium
                  |
     Approachable |  Aspirational
                  |
             Progressive
```

Place the brand in two positions:

- **Current position**: where stakeholders and users perceive the brand today
- **Target position**: where the brand needs to land

The gap between these two points defines the work.

### Positioning Statement Template

```
For [audience], [brand] is the [category] that [key differentiator],
unlike [competitor/alternative] which [competitor weakness].
We earn this position through [proof points].
```

Write the statement in one pass. Refine once. Do not workshop endlessly.

## 3. Brand Personality Spectrum

Define personality on four axes. Place a marker on each spectrum.

```
Formal      |----|----|----|----|----|  Casual
Premium     |----|----|----|----|----|  Accessible
Bold        |----|----|----|----|----|  Restrained
Warm        |----|----|----|----|----|  Cool
Playful     |----|----|----|----|----|  Serious
Classic     |----|----|----|----|----|  Modern
```

### How Personality Maps to Design

| Axis Position | Type Direction | Color Direction | Spacing | Motion |
|---|---|---|---|---|
| Formal | Serif, tight tracking | Muted, restrained palette | Generous whitespace | Subtle, slow |
| Casual | Sans-serif, open tracking | Saturated, varied palette | Compact, conversational | Bouncy, fast |
| Premium | Thin weights, large scale | Dark or monochrome base | Generous | Minimal, precise |
| Accessible | Medium weights, readable sizes | Bright, friendly palette | Functional | Purposeful |
| Bold | Heavy weights, large type | High contrast, saturated accent | Tight, dense | Dramatic |
| Restrained | Light weights, modest scale | Low contrast, neutral base | Airy | Nearly invisible |

Use this table to translate personality decisions into concrete design direction.

## 4. Direction Exploration

Develop 2-3 distinct visual directions before committing. Each direction must be meaningfully different — not variations on one idea.

### Direction Template

```
Direction Name: [Short evocative name, e.g., "Quiet Authority"]

Thesis: [One sentence. What this direction believes about the brand.]

Mood: [3-5 reference words, e.g., "editorial, confident, restrained, warm"]

Type Direction:
  Display: [family or style]
  Body: [family or style]
  Character: [what the pairing communicates]

Color Direction:
  Base: [light/dark, warm/cool neutral]
  Accent: [primary accent description]
  Range: [narrow/broad palette, how many colors]

Spatial Character:
  Density: [airy / balanced / compact]
  Grid: [wide margins / edge-to-edge / asymmetric]
  Rhythm: [even / varied / dramatic]

Signature Element:
  [One distinctive design move that makes this direction recognizable.
   Examples: oversized type, visible grid lines, rounded containers,
   monospaced details, full-bleed photography, geometric dividers.]

Strengths: [Why this direction works for the brand]
Risks: [Where this direction could fail or feel wrong]
```

Produce exactly 2-3 directions. Do not produce more. Each should solve the positioning gap differently.

## 5. Stakeholder Alignment Artifacts

Present these artifacts to get buy-in before moving to execution:

1. **Brand positioning statement** — one paragraph
2. **Personality profile** — the spectrum chart with markers placed
3. **Visual direction comparison** — all 2-3 directions side by side with key differences highlighted
4. **Recommendation** — which direction to pursue, with rationale tied back to positioning and audience

### Comparison Format

```
|                    | Direction A        | Direction B        | Direction C        |
|--------------------|--------------------|--------------------|--------------------|
| Thesis             |                    |                    |                    |
| Type character      |                    |                    |                    |
| Color character     |                    |                    |                    |
| Spatial feel        |                    |                    |                    |
| Signature element   |                    |                    |                    |
| Best for            |                    |                    |                    |
| Risk                |                    |                    |                    |
```

### Recommendation Statement Template

```
We recommend Direction [X] because it [solves the positioning gap by...],
[aligns with audience expectations around...],
and [differentiates from competitors who...].

Direction [Y] was strong but [specific limitation].
```

## 6. Design Principles

Distill 3-5 principles that guide every design decision in the system. Derive these from the chosen direction and positioning — not from generic best practices.

### Principle Format

```
Principle: [Name]
Definition: [One sentence explaining what it means]
Do: [Concrete example of the principle applied correctly]
Don't: [Concrete example of violating the principle]
```

### Example

```
Principle: Earned Confidence
Definition: Let quality speak through restraint, not decoration.
Do: Use generous whitespace and a single accent color to draw focus.
Don't: Add gradients, shadows, and multiple type styles to make something feel "premium."
```

Rules:

- Keep principles to 3-5. More than five means none are memorable.
- Each principle must be specific enough to settle a design argument.
- Avoid principles that any brand could claim (e.g., "clean and modern").
- Test each principle: if the opposite is also a valid design choice, the principle has teeth.

## Phase Gate

Do not proceed to token definition until:

- Positioning statement is written
- Personality spectrum is filled out
- One direction is selected with rationale
- Design principles are defined
- All artifacts are documented in the deliverable
